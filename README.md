# JavaScript Core Concepts – Project-Based Learning

This repository contains my JavaScript learning journey through small, focused projects. Each project is designed to understand one or more core JavaScript concepts by building and experimenting, without using any frameworks.

---

##  Repository Structure

Each folder represents a single concept-focused project.

### js-projects
- variables-basics
- mini-calculator
- guess-the-number
- ... to be continued


Inside each project folder:
- index.html – structure
- style.css – basic styling
- script.js – JavaScript logic
- notes.md – learnings and interview notes

---

## Concepts Covered

- Variables, data types, operators
- Functions and scope
- Conditionals and loops
- Arrays and objects
- DOM manipulation
- Events and event delegation
- Form handling and validation
- Asynchronous JavaScript (fetch, async/await)
- Closures and debouncing
- Error handling
- Immutability and state management

---

##  How to Run the Projects

1. Clone the repository
2. Open any project folder
3. Open `index.html` in a browser
(No build tools or dependencies required)
